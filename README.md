# RSign
RSign é um utilitário local para Windows desenvolvido em Delphi para assinatura de arquivos compatíveis com Authenticode (.exe, .dll, .msi, .cab, .cat). Suporta certificado PFX autoassinado, validação completa, verificação pós-assinatura, logs detalhados e arquitetura preparada para futuros certificados de Code Signing.
